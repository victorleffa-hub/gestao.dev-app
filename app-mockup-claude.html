#247 CRITICAL

Fix auth redirect loop

O redirect loop acontece quando o cookie SameSite nao esta configurado corretamente no middleware de autenticacao. O browser rejeita o cookie em cross-origin requests, causando um loop infinito entre /login e /callback.

Passos para reproduzir:
1. Limpar cookies do browser
2. Acessar qualquer rota protegida
3. Observar redirect loop no Network tab

Produto SaaS Aprovada

Transformar gestao.dev em Multi-tenant

Criado por Victor Leffa · 15 Mar 2026

Problema

O sistema gestao-dev funciona hoje como single-tenant. Cada empresa precisa de uma instalacao separada, dificultando escala e manutencao.

Solucao Proposta

Implementar multi-tenancy com shared DB usando org_id em todas as tabelas. Adicionar sistema de billing com Stripe + Pagar.me para cobranca por plano e seats.

Requisitos

Schema de organizations, membros, convites. Middleware de org resolution. Feature gating por plano. Billing UI completa.

Metricas de Sucesso

5 organizacoes pagantes nos primeiros 3 meses. Churn abaixo de 5%. MRR de R$2.000 no Q2.

Riscos

Complexidade da migracao de dados existentes. Performance de queries com filtro org_id em tabelas grandes.
